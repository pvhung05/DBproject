## Tổng quan

## Technologies Used
- **Frontend:**
  - React.js
  - Styled with CSS

- **Backend**
  - Spring Boot

- **Database**
  - MySql
  - 6 bảng: Employee, Food, Household, Invoice, InvoiceItem, Product.
